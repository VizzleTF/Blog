Данная политика описывает, какую информацию собирает сайт **onlyyaml.dev** и как она используется. Автор сайта — **Ivan K**.

#### Аналитика и cookies

Сайт использует следующие инструменты:

- **Yandex.Metrika** — счётчик посещений. Собирает анонимизированные данные о сеансах (браузер, ОС, страницы, источники трафика) в соответствии с [политикой Яндекса](https://yandex.ru/legal/confidential/). Использует cookies.
- **Umami Analytics** — self-hosted аналитика, собирает анонимные данные без cookies.
- **Cloudflare Web Analytics** — анонимная аналитика от Cloudflare, Inc. без cookies, в соответствии с [политикой Cloudflare](https://www.cloudflare.com/privacypolicy/).

Вы можете отключить cookies в настройках браузера — это не нарушит работу основного контента сайта.

#### Комментарии (Giscus)

Для комментариев используется **Giscus** на основе GitHub Discussions. Для публикации комментария требуется авторизация через GitHub. Данные хранятся на серверах GitHub в соответствии с [политикой конфиденциальности GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

#### Сторонние ссылки

Сайт содержит ссылки на внешние ресурсы (GitHub, Telegram, документация). Автор не несёт ответственности за политики конфиденциальности этих сайтов.
